# Simple Interest Calculator

## Overview
This project calculates **Simple Interest (SI)** based on the principal amount, rate of interest, and time period.

## Formula
Simple Interest is calculated using the formula:

SI = (Principal × Rate × Time) / 100

Where:
- **Principal (P)** = Initial amount of money
- **Rate (R)** = Interest rate per year
- **Time (T)** = Time period in years

## Features
- Calculates simple interest instantly
- Easy to understand formula
- Useful for basic financial calculations

## Example
If:
- Principal = 1000
- Rate = 5%
- Time = 2 years

Then:

SI = (1000 × 5 × 2) / 100 = 100

Total Amount = Principal + SI = **1100**

## Usage
1. Enter Principal amount.
2. Enter Rate of Interest.
3. Enter Time period.
4. The program calculates the Simple Interest.
